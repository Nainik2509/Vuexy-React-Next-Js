### Steps
<ul className='pl-3'>
  <li>

    </p>
  </li>
  <li>
    <p>
      Replace the <strong>layouts</strong> folder with the new version's <strong>layouts</strong> folder.
    </p>
  </li>
  <li>
    <p>
      Replace the <strong>themeConfig.js</strong> file with the new version's <strong>themeConfig.js</strong> file.
    </p>
  </li>
  <li>
    <p>
      Replace the <strong>router</strong> folder with the new version's <strong>router</strong> folder.
    </p>
  </li>
  <li>
    <p>
      Replace the <strong>App.js</strong> file with the new version's <strong>App.js</strong> file.
    </p>
  </li>
  <li>
    <p>
      Replace the <strong>index.js</strong> file with the new version's <strong>index.js</strong> file.
    </p>
  </li>
  <li>
    <p>
      Replace the <strong>package.json</strong> file with the new version's <strong>package.json</strong> file.
    </p>
  </li>
</ul>
