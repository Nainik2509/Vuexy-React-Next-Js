You might want to logout a user when the access token expires.
Here's how you can achieve that.

1. <p>Copy `PrivateRoute.js` somewhere out of `@core` folder.</p>
2. <p>Replace the PrivateRoute import in `src/router/routes` with yours.</p>
3. <p>Open the copied `PrivateRoute.js`.</p>
4. <p>Import jwt:</p>

```jsx
// ** JWT Imports
import jwt from 'jsonwebtoken'
```
5. <p>Create a isTokenExpired function with modifications according to your structure:</p>

```jsx
const isTokenExpired = token => {
  if (token) {
    return jwt.verify(token, 'SECRET', (err, decoded) => {
      return !Boolean(decoded)
    })
  } else {
    return true
  }
}
```

6. <p>In PrivateRoute component add the following statement just above `route.meta` condition</p>
```jsx
if (isTokenExpired(accessToken) {
    yourLogoutFunction()
    return <Navigate to='/login' />
}
```
