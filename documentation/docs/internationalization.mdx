<p>In this page you will find how to add/update Internationalization.</p>

### Overview

---

<p className='mb-5'>
  We're using{' '}
  <a href='https://www.i18next.com/' target='_blank'>
    i18next
  </a>{' '}
  for Internationalization. You can find our intl configuration in <code>src/configs/i18n</code>
</p>

### Adding new translations

---

<p>
  <code>public/assets/data/locales</code> folder contains template's core intl translations files.
</p>
<p className='mb-5'>You can add the your translations in these files.</p>

### Adding new languages

---

<p>Right now there are only four languages configured in the template.</p>
<p>Follow these steps to add new languages:</p>

<ol className='pl-3'>
  <li>
    Add new <code>YOUR_LANG.json</code> file in <code>public/assets/data/locales</code>.
  </li>
  <li>
    Thats it you can use <code>i18n.changeLanguage</code> in your app to change the language.
  </li>
</ol>
