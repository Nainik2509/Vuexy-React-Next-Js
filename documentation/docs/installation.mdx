### Guide

---

<p>
  First of all, make sure you have installed{' '}
  <a href='https://nodejs.org/en/' target='_blank'>

  </a>{' '}
  and{' '}
  <a href='https://www.npmjs.com/' target='_blank'>
    npm
  </a>{' '}
  /{' '}
  <a href='https://yarnpkg.com/' target='_blank'>
    yarn
  </a>
</p>
<Alert className='my-4' color='success'>
  <h4 className='alert-heading'>Tip</h4>
  <div className='alert-body'>Yarn package manager is recommended</div>
</Alert>
<Alert color='warning'>
  <h4 className='alert-heading'>Warning</h4>
  <div className='alert-body'>
   Please make sure you use the nodeâ€™s LTS version which is recommended by the official node site and not the one with the latest features or previous versions.

<Alert color='danger'>
  <h4 className='alert-heading'>Important</h4>
  <div className='alert-body'>

  </div>
</Alert>
<Alert color='danger'>
  <h4 className='alert-heading'>Important</h4>
  <div className='alert-body'>

    <strong>npm install --force --legacy-peer-deps</strong>
  </div>
</Alert>
<ul className='mb-3'>
  <li>After downloading zip from ThemeForest, unzip it in your desired location.</li>
  <li>
    In uncompressed folder you will find folder named <code>vuexy-admin-vX.X.X</code>. This will contain all versions of
    Vuexy Template. Move to <code>react-version</code> folder and open this folder in console / terminal.
  </li>
  <li>Run below command in console:</li>
</ul>

```bash
# For Yarn (Highly Recommended)
yarn

# For npm
npm install --force --legacy-peer-deps
```

<Alert className='my-4' color='success'>
  <h4 className='alert-heading'>Tip</h4>

      getting support
    </a>{' '}
    for our product.
  </div>
</Alert>

<ul className='mb-3'>
  <li>Now to run development server of Vuexy, run below command:</li>
</ul>

```bash
# For Yarn
yarn start

# For npm
npm run start
```
